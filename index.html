Janesville Dam Flood Gate Automation

Project Context

My goal in this project was to replace the existing manual winch system to move the four, 2 ton flood gates at the janesville dam with an automatic system.

Final Product

At the end of the eight month, $80,000 project, I had designed and oversaw the install of all the industrial, mechanical, and electrical components for a leadscrew-powered automatic gate actuation system. I also programmed the PLC and HMI that runs the system.

Control Cabinet

The controls for this project included a main PLC, four motor drivers with current sensors, four encoders, four limit switches, HMI and physical controls, data storage, and light and noise alarms for all movement.

Motor and Screw Install

Pictured here is the first full motor, screw, and mounting platform to be installed. The existing infrastructure was not made with leadscrews in mind, so there was no existing platforms to mount on. The solution was to use fast and cheap laser cut mounting plates to create the nessesary platform for minimal cost.

Mecanical System

This is one end of the power train that drives the gates up and down. The encoder on the oposite end of the motor allows for precise tracking of the position of the gate, so the system knows whether it is fully up or down, or somewhere in between. However, an upper limit switch was used on each gate as a failsafe, and the drives also monitor current as a failsafe against going below fully down.
